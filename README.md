# AI-Grading-System
Production-ready AI Exam Grader built with LangGraph, FastAPI &amp; Streamlit. Uses Llama 4 Vision to parse handwritten PDFs &amp; Llama 70B for semantic, partial-credit grading. Features async batching, PyMuPDF slicing, SQLite caching, and a decoupled architecture. Evaluates logical context, not just exact string matches.
